Protected Languages

The following programming languages are protected under this act:

  • Scratch
  • Snap!
  • JavaScript
  • PHP
  • Python

Users of these languages must comply with the law. Violations may result in fines, jail time, or criminal charges.

Unprotected Languages

The following languages are considered unprotected:

  • HTML
  • CSS
  • XML
  • Java
  • JSON

These languages are exempt from protection and may be used freely, though responsible practices are encouraged.

Prohibited Activities

The following actions are strictly prohibited under this act for all protected languages:

  • Obfuscating code to hide malicious intent
  • Hacking or unauthorized access
  • Abusing system resources or users
  • Crashing or destabilizing software or networks
  • Writing or distributing malicious code

Violating these rules may lead to imprisonment in addition to fines and criminal penalties.

Terms of Use

  • By using protected languages, you agree to comply with this Act.
  • False violation reports are strictly prohibited and will result in penalties.
  • You accept full responsibility for the code you write and distribute.
  • The Act applies globally, regardless of location or platform.
